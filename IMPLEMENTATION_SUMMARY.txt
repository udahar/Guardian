## Configuration Defaults

All settings are configurable in `config.py`:

```python
docker_cache_prune_threshold_gb: float = 3.0     # Auto-prune when > 3GB
docker_vhd_bloat_alert_gb: float = 4.0          # Alert when > 4GB
docker_images_warn_gb: float = 10.0             # Warning when > 10GB
docker_auto_prune: bool = True                   # Enable auto-pruning
```

Daemon.json settings (applied by setup):
```json
{
  "log-driver": "json-file",
  "log-opts": {
    "max-size": "10m",
    "max-file": "3"
  }
}
```

## Verification Steps

```bash
curl http://localhost:4001/docker/config-summary
```

Look for:
- `log_rotation_enabled: true`
- `max_log_size: "10m"`
- `max_log_files: 3`

### 2. Manual Cleanup Test
```bash
curl -X POST http://localhost:4001/docker/prune -d '{"aggressive": false}'
```

Should return:
- `success: true`
- `space_freed_gb: X.XX`
- List of actions performed

### 3. Scheduled Task Created
**Windows:**
```bash
schtasks /query /tn "Guardian\GuardianDockerCleanup"
```

**Linux:**
```bash
crontab -l | grep Guardian
```

**macOS:**
```bash
launchctl list | grep guardian
```
